void finish() ¶
Finalize the zlib stream.
This causes zlib to flush any pending write data and write end-of-stream information, including the checksum. Once finish() has been called, no new data can be written to the stream.
void verifyChecksum() ¶
Verify the checksum at the end of the zlib stream (by finish()).
May only be called after all data has been read.

Throws:

TTransportException when the checksum is corrupted or there is still unread data left.
