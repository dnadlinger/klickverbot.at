thrift.server.base

class TServer ¶
Base class for all Thrift servers.
By setting the eventHandler property to a TServerEventHandler implementation, custom code can be integrated into the processing pipeline, which can be used e.g. for gathering statistics.
void serve(TCancellation cancellation = null) [abstract] ¶
Starts serving.
Blocks until the server finishes, i.e. a serious problem occured or the cancellation request has been triggered.

Server implementations are expected to implement cancellation in a best- effort way – usually, it should be possible to immediately stop accepting connections and return after all currently active clients have been processed, but this might not be the case for every conceivable implementation.
